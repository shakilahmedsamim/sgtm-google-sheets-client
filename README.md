# Google Sheets Webhook Client for Server GTM

Custom Server-side Google Tag Manager (sGTM) Client Template for receiving webhook events from Google Sheets.

## Features

- Google Sheets Webhook Support
- Purchase Event Creation
- Dynamic Transaction ID
- Dynamic Purchase Value
- Dynamic Currency
- Dynamic Product Items
- Dynamic FBP
- Dynamic FBC
- Dynamic User Agent
- Dynamic IP Address
- Meta Conversions API Ready

## Data Source

Google Sheets → Apps Script → Server GTM

## Supported Platforms

- Google Sheets
- Google Apps Script
- Server GTM
- Meta Conversions API

## Installation

1. Create a Custom Client Template in Server GTM
2. Add the template code
3. Configure a webhook endpoint path
4. Publish the container
5. Send POST requests from Google Sheets Apps Script
